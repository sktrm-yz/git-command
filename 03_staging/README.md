# ステージング＆コミット

### ステージングエリアへ追加

```
git add [file_name]
```

### ステージングを取り消し

```
git reset
```

### コミット

```
git commit -m "[commit_message]"
```

### コミット取り消し

```
git reset --soft HEAD^
```

### コミット履歴確認

```
git log
```

### 差分確認

ステージングエリア

```
git diff --staged
```

### 差分確認

```
git diff
```
